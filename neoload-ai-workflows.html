🤖

확장된 MCP 기능

자연어 프롬프트로 테스트 인프라 관리와 테스트 분석을 가속화. NeoLoad Web V2의 분석 트리뷰, 시계열 추적, 실시간 그래프 생성으로 인사이트를 빠르게 도출한다.
